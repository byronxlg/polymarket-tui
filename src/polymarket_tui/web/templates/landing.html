// what it does

Everything Polymarket, one keystroke away

The whole surface of the exchange, rendered as a dense, fast, keyboard-native interface. No mouse required.

📊

Browse every market

Trending, Politics, Sports, Crypto and more, sorted by volume or liquidity. Search markets and traders with /.

📈

Live order books

Real-time YES/NO books and the trade tape stream straight from the CLOB. Flip sides with space.

📉

Price charts

Full price history for any market or outcome, inline. Move the cursor to inspect any point in time.

💰

Portfolio & P&L

Positions, cash, and profit history for your wallet, or observe any public trader's book read-only.

⚡

Place orders

Limit and market orders under the live book, priced in cents. Decimal-exact, validated, audited to a local log.

🔒

Safe by default

Every session opens in DRY mode: orders are signed but never posted. Going LIVE is an explicit, per-session opt-in.

// one scheme everywhere

Built for the keyboard

A small core of keys, one meaning each. Arrows move and drill; space is always the contextual toggle. Press ? in the app for the full reference.

↑↓←→Move, drill in with right/enter, step out with left/esc
spaceContextual toggle: star, follow, flip book, flip BUY/SELL
tabCycle the screen's selector: category, timeframe, pane
bsBuy / sell entry under the live book, price in cents
/Search markets and traders
pwPortfolio and watchlist
LToggle DRY / LIVE for the session (confirmed)
?qFull key reference / quit

Run it your way

Install the CLI locally, or just launch it in the browser, no install required.

curl
curl -sSL https://raw.githubusercontent.com/byronxlg/polymarket-tui/main/install.sh | bash
brew
brew tap byronxlg/polymarket-tui https://github.com/byronxlg/polymarket-tui
brew install polymarket-tui
source
git clone https://github.com/byronxlg/polymarket-tui
cd polymarket-tui && uv sync && uv run polymarket-tui
🌐

Or run it right here

The full TUI, streamed to your browser over a terminal, no local setup. Each visitor gets an isolated session that starts in DRY mode.
